# @pompcore/types

Shared TypeScript types and contracts for the PompCore platform.

## Overview

Zero-dependency package containing all shared type definitions, domain contracts, permission mappings, and pure utility functions used across PompCore services.

## Install

```bash
pnpm add @pompcore/types
```

## Usage

```ts
import type { UserRole, ServiceCode } from '@pompcore/types';
```

## Build

```bash
pnpm --filter @pompcore/types build
```
