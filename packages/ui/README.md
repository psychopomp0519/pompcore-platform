# @pompcore/ui

Nebula Design System — shared UI components, icons, hooks, and utilities for the PompCore platform.

## Overview

Provides glassmorphism-based components (GlassCard, Modal, Button, etc.), RPG-themed SVG icons, theme management, and common utilities (date, currency, error messages).

## Install

```bash
pnpm add @pompcore/ui
```

## Usage

```tsx
import { GlassCard, Button, LoadingSpinner, useThemeStore } from '@pompcore/ui';
import '@pompcore/ui/styles/base.css';
```

## Build

```bash
pnpm --filter @pompcore/ui build
```
