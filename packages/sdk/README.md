# @pompcore/sdk

Platform configuration factory for the PompCore ecosystem.

## Overview

Provides environment-aware configuration setup, service registration, and shared platform utilities used by all PompCore apps.

## Install

```bash
pnpm add @pompcore/sdk
```

## Usage

```ts
import { createPlatformConfig } from '@pompcore/sdk';
```

## Build

```bash
pnpm --filter @pompcore/sdk build
```
