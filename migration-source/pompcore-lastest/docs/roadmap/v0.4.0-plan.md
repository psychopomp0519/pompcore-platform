# v0.4.0 계획서: 탭 기반 간결한 디자인 리뉴얼

**상태**: 승인 대기 → 승인됨 (다음 세션에서 실행)
**참고 디자인**: `docs/design/pompcoredesign2.svg`

---

## 구현 목적

pompcoredesign2 참고, 탭 네비게이션 기반의 간결한 싱글페이지 레이아웃으로 전환.
최대한 간결하고 탭을 통해서 자세한 내용을 확인할 수 있는 형태.

## 디자인 방향 (pompcoredesign2 분석)

- **다크 기반**: 깊은 네이비/슬레이트 배경
- **간결한 레이아웃**: 상단에 로고 + 탭형 네비게이션 (Apps, About, Careers, Contact)
- **탭 기반 콘텐츠**: Vault와 Quest 카드가 나란히 배치, 각각 미리보기 UI 표시
- **기하학적 장식**: 삼각형 모양의 배경 장식 요소
- **미니멀**: 최소한의 텍스트, 카드 클릭으로 상세 내용 확인하는 구조

## 주요 변경 파일

| 수정 파일 | 변경 내용 |
|-----------|-----------|
| `src/pages/Home/Home.tsx` | 탭 기반 레이아웃 (프로젝트/공지사항/패치노트/소개를 탭으로 전환) |
| `src/pages/Home/HeroSection.tsx` | 간결한 히어로 → 로고 + 한 줄 카피 + 프로젝트 카드 2열 |
| `src/components/layout/Header.tsx` | 탭 네비게이션으로 전환 (페이지 이동 대신 탭 전환) |
| `src/components/project/ProjectCard.tsx` | 디자인2 스타일 카드 (미리보기 UI 포함) |

## 모듈 간 데이터 흐름

- Header 탭 클릭 → Home 컴포넌트 상태 변경 → 해당 섹션 렌더링
- 별도 페이지(/projects, /announcements 등)는 유지하되, 홈에서 탭으로도 접근 가능

## 예상 문제점

- 탭 전환과 라우터 페이지 이동 간의 UX 일관성 유지 필요
- 탭 내 콘텐츠 양이 많아지면 스크롤 vs 페이지네이션 결정 필요
