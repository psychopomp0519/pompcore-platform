# 비즈니스 전략

> 최종 목표: **구글 검색을 통한 유기적 트래픽 증가**

## 수익화

### AdSense
- Google AdSense 연동으로 광고 수익 창출
- 사용자 경험을 해치지 않는 위치에 배치
- 콘텐츠 품질 유지로 AdSense 승인 기준 충족

### Polar 구독 시스템
- Polar 연동을 통한 구독(후원) 시스템 도입
- 프리미엄 기능 또는 후원 기반 모델 검토

## 사용자 분석

### Microsoft Clarity
- Clarity 연동으로 사용자 행동 분석
- 히트맵, 세션 리플레이 등 활용
- 데이터 기반 UX 개선

## 검색 최적화

### SEO (Search Engine Optimization)
- 메타 태그 최적화 (title, description, OG 이미지)
- 시맨틱 HTML 구조
- sitemap.xml, robots.txt 설정
- 구조화된 데이터 (JSON-LD)
- Core Web Vitals 최적화

### GEO (Generative Engine Optimization)
- AI 검색엔진 (Google SGE, Bing Chat 등) 대응
- 명확하고 구조화된 콘텐츠 작성
- FAQ, How-to 등 AI가 인용하기 좋은 형식 활용

## 구현 우선순위
1. SEO 기본 설정 (메타 태그, sitemap, robots.txt)
2. Microsoft Clarity 연동
3. AdSense 신청 및 연동
4. Polar 구독 시스템 연동
5. GEO 최적화 (콘텐츠 구조화)
