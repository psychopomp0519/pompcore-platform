# PompCore — 프로젝트 정체성 문서

> **"일상을 플레이하다 — Play Your Day"**
>
> PompCore는 서브컬쳐 게임의 몰입감과 감성을 일상 도구에 접목하여,
> 사용자가 자신의 삶을 하나의 게임처럼 능동적으로 플레이할 수 있는 **통합 생활 플랫폼**이다.

---

## 1. 프로젝트 한 줄 소개

**가계부, 일정 관리 등 생활 필수 도구를 서브컬쳐 게임 감성 UI로 제공하는 통합 라이프스타일 플랫폼.**

기존의 생산성 도구들이 "딱딱하고 지루하다"는 문제를 해결한다. PompCore는 원신, 명일방주, 블루아카이브 같은 서브컬쳐 게임의 미학(다크 테마, 네온 글로우, 판타지 모티프)을 차용하되, 실제로 사용자의 삶에 도움이 되는 도구를 만든다. 이것이 PompCore가 말하는 **"실용적인 판타지"**다.

---

## 2. 비전 & 미션

### 비전 (Vision)

서브컬쳐 게임의 몰입감과 감성을 일상 도구에 접목하여, 사용자가 자신의 삶을 하나의 게임처럼 능동적으로 플레이할 수 있는 통합 생활 플랫폼을 구축한다.

### 미션 (Mission)

| 미션 |
|---|
| 생활 필수 도구를 서브컬쳐 게임 감성 UI로 제공한다 |
| 무료 사용자도 핵심 기능에 불편함 없이 접근할 수 있도록 보장한다 |
| AI 네이티브 개발 방식을 채택하여 소규모 팀으로도 고품질 서비스를 빠르게 구축한다 |
| 사용자 피드백을 적극 수용하여 함께 성장하는 서비스를 만든다 |

## 4. 브랜딩 & 디자인 아이덴티티

### 4.1 Nebula(성운) 테마

PompCore의 디자인 시스템은 **"Nebula(성운)"**를 테마로 한다. 서브컬쳐 게임에서 자주 사용되는 딥 다크 배경 위에 네온 글로우 효과를 강조하는 방식이다.

#### 컬러 팔레트

| 컬러명 | HEX | 역할 | 사용처 |
|---|---|---|---|
| Violet (Primary) | `#7C3AED` | 브랜드 메인 | 로고, 버튼, 네비게이션 활성 |
| Gold (Accent) | `#FFD700` | 강조/보상 | 경험치, 달성, 글로우 효과 |
| Pink (Highlight) | `#EC4899` | 하이라이트 | 경고, 알림, 포인트 컬러 |

#### 다크 모드 배경 (3단계)

| 레벨 | HEX | 용도 |
|---|---|---|
| dark-1 | `#0C0818` | 앱 전체 배경 |
| dark-2 | `#110D20` | 카드/섹션 배경 |
| dark-3 | `#150F28` | 오버레이/모달 |

#### 라이트 모드

- 배경: 푸른 하늘 그라디언트 (sky palette 7단계)
- 장식: 복합 구름 시스템 (가장자리+코어 2레이어 입체 구름, 드리프트 애니메이션)

#### 다크 모드

- 배경: 밤하늘 + 별 파티클 (twinkle 애니메이션)
- 효과: 네온 글로우, 글래스모피즘 카드

### 4.3 슬로건

> **"일상을 플레이하다"** — Play Your Day

"플레이"가 게임과 일상의 이중 의미를 자연스럽게 담아내며, 서비스 확장 시에도 일관된 메시지를 유지할 수 있다. 영문 전환도 용이하여 글로벌 확장에 유리하다.

### 4.4 기업 이미지

PompCore는 **"실용적인 판타지"**를 지향한다. 판타지 RPG(Genshin Impact 스타일) 분위기로 브랜드를 차별화하되, 실제로 사용자의 삶에 도움이 되는 도구를 만든다. Violet 기반 팔레트가 게이미피케이션 컨셉에 부합하며, 19종 커스텀 SVG 아이콘으로 일관된 비주얼을 유지한다.

## 5. 타겟 사용자

### 1차 타겟

서브컬쳐(애니, 게임, 라이트노벨 등)에 관심이 높은 **10대 후반~20대 중반**. 디지털 네이티브로서 새로운 서비스 수용도가 높고, SNS를 통한 자발적 확산이 활발하다.

### 페르소나

| 이름 | 나이 | 특징 | 니즈 |
|---|---|---|---|
| **유진** (대학생) | 22세 | 블루아카이브, 명일방주 유저. 자취 중 | 기존 가계부가 "너무 딱딱해서" 꾸준히 못 씀. 게임 감성으로 지출 기록하고 싶다 |
| **민수** (취준생) | 25세 | 원신, 스타레일 유저. 취준+알바 병행 | 일정, 가계부, 할 일을 한 곳에서 관리하고 싶다 |
| **하은** (고등학생) | 18세 | 서브컬쳐 전반 관심. 용돈 관리 시작 | 귀여운 UI와 보상 시스템이 있으면 꾸준히 사용할 의향 |

## 6. 서비스 라인업

### 6.1 메인 사이트 (pompcore.com)

PompCore의 브랜딩 허브이자 SSO 인증센터. 모든 서브 프로젝트의 관문 역할을 한다.

- 랜딩 페이지 (6섹션: Hero → Services → Why → Upcoming → FAQ → CTA)
- 프로젝트 소개 / 회사 소개 / 패치노트 / 공지사항
- 통합 인증 (Supabase 기반 SSO, Google OAuth)
- 팀원 모집 시스템

### 7.3 AI 네이티브 개발 철학

PompCore는 **AI 네이티브 개발 방식**을 채택한다. 프로젝트 기획/계획 확정 후 AI 코딩 도구에 입력할 프롬프트를 설계하는 것이 핵심 업무다. 이를 통해 소규모 팀(현재 1인)으로도 고품질 서비스를 빠르게 구축한다.

#### 개발 프로세스

```
계획서 작성 → 승인 대기 → 코드 작성 → 패치노트 기록 → 문서 업데이트
```

#### 코딩 컨벤션 (3원칙 + 2보안)

1. **극단적 모듈화** — 기능/UI/데이터/API 목적별 파일 분리, 모듈 간 의존성 최소화
2. **방어적 프로그래밍** — 엄격한 타입, 명시적 예외 처리
3. **가독성 최우선** — 파일 최상단 역할 주석, 핵심 함수 JSDoc
4. **환경변수 분리** — API 키, DB 비밀번호 등 민감 정보는 `.env`로 분리
5. **LSP 친화적 코드** — 명시적 타입, named export, `any` 금지

## 8. 수익화 구조

### 핵심 원칙

> **"무료 사용자가 불편함을 느끼지 않아야 한다."**

### 8.1 광고 수익 (Google AdSense)

- 디스플레이 광고 기반 수익 창출
- 핵심 기능 UI 위에는 절대 광고를 배치하지 않음
- 유료 구독 시 광고 제거

### 8.2 부분유료화 (Freemium)

| 무료 (기본) | 프리미엄 (유료 구독) |
|---|---|
| 모든 핵심 기능 이용 가능 | 광고 제거 |
| 기본 테마 제공 | 프리미엄 테마 & 커스터마이징 |
| 기본 리포트 & 차트 | 고급 분석 리포트 & 데이터 내보내기 |
| 기본 알림 | 고급 알림 커스터마이징 |
| 기본 배지/칭호 | 한정 배지, 특별 칭호, 프로필 꾸미기 |

### 8.3 구독 시스템 (Polar)

Polar 연동을 통한 구독(후원) 시스템. 프리미엄 기능 또는 후원 기반 모델.

### 8.4 SEO/GEO를 통한 오가닉 트래픽

- **SEO**: 메타 태그, sitemap.xml, robots.txt, JSON-LD 구조화 데이터, Core Web Vitals
- **GEO**: AI 검색엔진(Google SGE, Bing Chat 등) 대응, FAQ/How-to 형식 콘텐츠
- **최종 목표**: 구글 검색을 통한 유기적 트래픽 증가

## 9. 팀 & 운영 방침

### 9.1 현재 팀 구조

현재 **1인 체제**로 운영 중. AI 네이티브 개발 방식으로 1인이 기획-디자인-개발-운영을 모두 담당한다.

### 9.2 모집 포지션

| 포지션 | 핵심 역할 |
|---|---|
| UI/UX 디자이너 | 서브컬쳐 게임 감성 디자인 시스템 구축 |
| 기획자 / PM | 서비스 기능 기획, AI 프롬프트 설계, 일정 관리 |
| 프론트엔드 개발자 | React 19 + TS 컴포넌트 개발, 게임 감성 인터랙션 |
| 백엔드 / Supabase 개발자 | 데이터 모델링, 인증, RLS 보안, 실시간 동기화 |

### 9.3 팀 확장 프로세스

```
아이디어 제출 → 검토 & 승인 → 팀 구성 → 개발 & 출시
```

합류 희망자가 자신만의 프로젝트 아이디어를 제출하면, PompCore의 비전/기술 스택 적합성을 검토하여 승인. 승인 후 아이디어 제안자가 팀을 꾸려 독립적으로 개발한다.

### 9.4 운영 방침

- **유저 피드백 적극 수용**: 인앱 피드백 시스템, 접수→분류 1일→검토 3일→반영 7일 이내
- **무료 우선 정책**: 유료 혜택은 "더 좋은 경험"에만 한정
- **데이터 & 개인정보**: 최소 수집 원칙, Supabase RLS로 데이터 격리, 투명한 개인정보 처리방침

## 부록: 프로젝트 확장 포인트

PompCore는 데이터 기반 자동 렌더링 구조를 채택하여, 최소한의 코드 수정으로 서비스를 확장할 수 있다.

| 확장 항목 | 수정 파일 | 방식 |
|---|---|---|
| 새 서브 프로젝트 추가 | `src/constants/projects.ts` | 배열에 항목 추가 → UI 자동 반영 |
| 새 페이지 추가 | `src/router/index.tsx` | 라우트 등록 |
| 새 아이콘 추가 | `src/components/icons/Icons.tsx` | ICON_MAP에 등록 |
| 새 브랜드 추가 | `src/components/common/BrandText.tsx` | BRAND_CONFIG에 등록 + 로고 SVG |
| 디자인 토큰 변경 | `tailwind.config.ts` | 중앙 관리, 일괄 변경 |
| 패치노트 추가 | `src/constants/patchnotes.ts` | 배열에 항목 추가 → 자동 반영 |
| 공지사항 추가 | `src/constants/announcements.ts` | 배열에 항목 추가 → 자동 반영 |
| FAQ 추가 | `src/constants/faq.ts` | 배열에 항목 추가 → 자동 반영 |
