# v0.2.0 패치노트

**날짜**: 2026-03-04
**주제**: 라이트 테마 기반 디자인 리뉴얼 + 다크 모드 토글

---

## 변경 사항

### 디자인 시스템 전환
- **라이트 테마 기본값 적용**: 기존 다크 전용 → 라이트 기본 + 다크 모드 지원
- **Navy 컬러 팔레트 추가**: 로고 색상(#2B3442) 기반 navy 컬러 시스템
- **Surface 컬러 재정의**: 라이트(#f8fafc), 다크(#0f172a) 배경색
- **카드 스타일 이원화**: 라이트(화이트 + 섀도우), 다크(글래스모피즘)
- **hover-lift 효과**: hover-glow 대체, 라이트 모드에 적합한 상승 효과

### 다크 모드 토글
- **themeStore (Zustand)**: 라이트/다크 상태 관리 + localStorage 영속화
- **Header 토글 버튼**: 🌙/☀️ 아이콘으로 즉시 전환
- **Tailwind dark: 접두사**: class 기반 다크 모드 활성화

### 로고 통합
- **SVG 로고 적용**: pompcore, vault, quest 로고 파일 assets/logos/ 배치
- **Header 로고**: pompcorelogo.svg 렌더링
- **ProjectCard logoSrc**: 프로젝트별 로고 이미지 표시 (이모지 폴백)

### 컴포넌트 업데이트 (라이트/다크 듀얼 모드)
- Button: secondary, ghost, outline 변형 듀얼 모드
- GlassCard: .card 클래스 기반, hover-lift 적용
- Header: 배경 투명도 분기, 네비게이션 색상 분기
- Footer: 라이트(화이트 배경), 다크(반투명 배경)
- Layout: bg-surface / bg-surface-dark 분기
- ProjectCard: 상태 배지 듀얼 모드, 호버 그라디언트 분기

### 페이지 업데이트
- HeroSection: 배경 오브 투명도 분기, 타이틀 navy 적용
- FeaturesSection: 카드 텍스트 듀얼 모드
- ProjectsPreview: 예고 카드 보더 듀얼 모드

### 프로젝트 구조
- **docs/templates/**: 프로젝트 이식용 템플릿 3종 추가
  - PROJECT_BOOTSTRAP.md, AI_WORK_RULES.md, DOCS_STRUCTURE.md
- **docs/design/**: 디자인 파일 저장소 생성
- **Project 타입 확장**: logoSrc 필드 추가
