# v0.1.0 - 프로젝트 기초 구조 설정

**날짜:** 2026-03-04

## 개발된 내용

### 기술 스택 구성
- React 19 + TypeScript 5.9 + Vite 7.3 기본 구조
- Tailwind CSS 3 + PostCSS + Autoprefixer 스타일링 시스템
- React Router DOM 라우팅
- Supabase 클라이언트 (Auth + DB)
- Zustand 상태관리

### 디자인 시스템
- PompCore 브랜드 컬러 팔레트 정의 (Indigo-Violet 기반)
- 글래스모피즘 카드, 그라디언트 텍스트, 호버 글로우 효과
- 반응형 레이아웃 (모바일 햄버거 메뉴 포함)
- 커스텀 애니메이션 (fade-in, slide-up, float)

### 공통 컴포넌트
- `Button` - variant(primary/secondary/ghost/outline), size(sm/md/lg), 로딩 상태 지원
- `GlassCard` - 글래스모피즘 카드, 호버 효과, 패딩 옵션
- `ProjectCard` - 프로젝트 정보 카드, 상태 배지, 링크 연결

### 레이아웃
- `Header` - 스크롤 반응형 글래스 배경, 모바일 메뉴
- `Footer` - 프로젝트 링크 자동 생성, 회사 정보
- `Layout` - Header + Content + Footer 래퍼

### 페이지
- `Home` - Hero 섹션 (그라디언트 오브 배경, CTA) + 프로젝트 미리보기
- `Projects` - 프로젝트 허브 (카드 그리드)
- `About` - 회사 소개 + 핵심 가치
- `Auth/Login` - 로그인 폼 (SSO 준비)
- `Auth/Register` - 회원가입 폼 (SSO 준비)

### 인증 시스템 (SSO 골격)
- Supabase 클라이언트 설정 (.env 분리)
- `authService` - login/register/logout/getCurrentUser/onAuthStateChange
- `authStore` (Zustand) - 전역 인증 상태 관리

### 프로젝트 데이터
- Vault (가계부) - emerald 컬러, coming_soon 상태
- Quest (일정관리) - blue 컬러, coming_soon 상태
- 프로젝트 추가는 `constants/projects.ts` 배열에 항목 추가만으로 완료

## 기존 대비 변경된 부분
- 기존 Vite 기본 템플릿 코드 제거 (App.css, index.css, 카운터 데모)
- Tailwind CSS 기반 글로벌 스타일로 전환
- SPA 라우팅 시스템 도입
