👋 안녕하세요! PompCore입니다.

"일상을 게임처럼, 관리를 모험처럼"

PompCore는 서브컬쳐 게임의 감성을 입힌 라이프 매니지먼트 플랫폼이에요.
가계부를 쓰는 게 RPG 인벤토리를 관리하는 느낌이라면?
일정을 짜는 게 퀘스트를 수락하는 느낌이라면?
그런 경험을 만들고 싶어서 시작한 프로젝트입니다.


----------------------------------------
🌟 우리가 만들고 있는 것
----------------------------------------

PompCore는 하나의 세계관 안에서 여러 서비스가 연결되는 구조예요.
서브컬쳐 게임 특유의 UI/UX를 공유하면서, 각 서비스가 하나의 "모듈"처럼 동작합니다.

[Vault] — 가계부
  돈의 흐름을 게임 속 자원 관리처럼.
  수입/지출을 직관적으로 파악하고, 절약 목표를 달성하면 보상을 받는 구조.

[Quest] — 일정 관리
  할 일과 일정을 퀘스트처럼 관리.
  데일리/위클리/메인 퀘스트 형태로 목표를 세우고 클리어해 나가는 경험.

[향후 확장] — 온라인 과외 플랫폼
  교육 콘텐츠를 게임화된 환경에서 제공하는 서비스.
  Vault와 Quest가 안정되면 본격적으로 개발 예정.


----------------------------------------
🛠️ 기술 스택
----------------------------------------

프론트엔드: React 19 + TypeScript + Vite
스타일링: Tailwind CSS 3
상태관리: Zustand
백엔드/DB: Supabase (Auth, Database, Storage, Realtime)
디자인 컨셉: 서브컬쳐 게임 감성 UI (다크 테마 기반, 네온/글로우 효과, 판타지 요소)


----------------------------------------
💡 우리의 작업 방식 — AI 네이티브 개발
----------------------------------------

PompCore는 조금 독특한 방식으로 개발해요.

팀원이 직접 코드를 한 줄 한 줄 작성하는 게 아니라,
"전반적인 계획을 확정하고, AI 코딩 도구에 넣을 프롬프트를 설계하는 것"이 핵심이에요.

즉, 우리가 집중하는 건:
  - 서비스 기획과 UX 설계
  - 기술적 의사결정과 아키텍처 설계
  - AI 도구가 정확한 결과를 내도록 프롬프트를 작성하고 검증하는 것
  - 결과물의 품질 관리와 통합

코딩 실력도 중요하지만, "무엇을 만들어야 하는지 명확하게 정의하는 능력"을
더 중요하게 생각합니다.


----------------------------------------
📢 모집 포지션
----------------------------------------

아래 4개 포지션에서 함께할 분을 찾고 있어요!


1) UI/UX 디자이너

  하는 일:
    - 서브컬쳐 게임 감성의 디자인 시스템 구축
    - Vault, Quest 등 각 서비스의 화면 설계 및 프로토타이핑
    - 일관된 세계관을 유지하는 컴포넌트/아이콘/일러스트 디자인

  이런 분이면 좋겠어요:
    - 서브컬쳐 게임(명일방주, 블루아카이브, 원신 등)의 UI에 관심이 많은 분
    - Figma 등 디자인 툴 사용 가능한 분
    - 다크 테마, 글로우 효과, 판타지 감성에 감각이 있는 분


2) 기획자 / PM

  하는 일:
    - 서비스 기능 기획 및 사용자 플로우 설계
    - AI 코딩 도구용 프롬프트 설계 및 품질 검증
    - 프로젝트 일정 관리 및 팀 커뮤니케이션 조율

  이런 분이면 좋겠어요:
    - 서비스 기획 경험이 있거나 관심이 많은 분
    - 사용자 관점에서 생각하고 문서로 정리하는 걸 좋아하는 분
    - AI 도구를 활용한 개발 프로세스에 호기심이 있는 분


3) 프론트엔드 개발자

  하는 일:
    - React 19 + TypeScript 기반 컴포넌트 개발 및 코드 리뷰
    - AI가 생성한 코드의 품질 검증 및 최적화
    - 게임 감성 인터랙션/애니메이션 구현

  이런 분이면 좋겠어요:
    - React + TypeScript 경험이 있는 분
    - Tailwind CSS로 스타일링 가능한 분
    - 멋진 인터랙션과 애니메이션에 욕심이 있는 분


4) 백엔드 / Supabase 개발자

  하는 일:
    - Supabase 기반 데이터 모델링 및 API 설계
    - 인증, 실시간 데이터, 스토리지 등 백엔드 인프라 구축
    - 보안 정책(RLS) 설계 및 성능 최적화

  이런 분이면 좋겠어요:
    - Supabase 또는 Firebase 등 BaaS 경험이 있는 분
    - PostgreSQL, REST API에 대한 이해가 있는 분
    - 서버리스 아키텍처에 관심이 있는 분
