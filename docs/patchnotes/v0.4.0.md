# v0.4.0 패치노트 (2026-03-05)

## Nebula 테마 디자인 리뉴얼 + 팀원 모집 시스템

### 디자인 시스템 전면 교체
- **Nebula 컬러 팔레트**: Indigo → Violet(#7C3AED) 기반, accent-gold(#FFD700), accent-pink(#EC4899)
- **Cinzel / Cinzel Decorative 폰트**: 타이틀용 display 폰트 추가
- **다크 모드 3단계 배경**: #0C0818, #110D20, #150F28
- **라이트 모드**: #FAF8FF(surface), #FFFFFF(cards)
- **text-gradient**: violet → gold → pink 그라디언트
- **section-divider-gradient**: 섹션 간 그라디언트 구분선

### 랜딩 페이지 6섹션 구조
- **HeroSection**: 의식형(Ceremonial) 스타일, 별 파티클(다크), 장식 프레임, 엠블럼, 서비스 미니 카드
- **ServicesSection**: Vault/Quest 대형 카드, ABILITY STATS 패널, 기능 리스트
- **WhySection**: 4열 핵심 가치 카드 (SVG 아이콘)
- **UpcomingSection**: 곧 만나볼 서비스 3열 카드, 알림 신청 CTA
- **FaqSection**: 아코디언 FAQ (grid-rows CSS 트랜지션 애니메이션)
- **CtaBanner**: Nebula 그라디언트 배경 CTA

### SVG 아이콘 시스템
- 16종 SVG 아이콘 컴포넌트 (Icons.tsx)
- DynamicIcon 공통 컴포넌트 (IIFE 패턴 제거)
- ICON_MAP 레지스트리로 동적 렌더링
- 사이트 전체 이모지 → SVG 교체

### 브랜드 텍스트 시스템
- BrandText 컴포넌트 (PompCore, Vault, Quest, Academy)
- PompCoreLogo 컴포넌트 (POMP + CORE 분리 표기)
- Cinzel 폰트 + 서비스별 고유 컬러

### 팀원 모집 시스템
- /recruit: 포지션 소개 + 지원서 양식 (localStorage 임시 저장)
- /recruit/admin: 관리자 지원서 열람 페이지 (비밀번호 인증)
- 4개 포지션: UI/UX 디자이너, PM, 프론트엔드, 백엔드

### 코드 품질 개선
- 관리자 비밀번호 .env 환경변수 분리
- HeroSection gradient 클래스 중복 제거
- BrandText 타입 시스템 개선 (string 확장)
- FaqSection 아코디언 CSS transition 애니메이션

### 신규/수정 파일
- 신규: Icons.tsx, BrandText.tsx, ServicesSection.tsx, WhySection.tsx, UpcomingSection.tsx, FaqSection.tsx, Recruit.tsx, RecruitAdmin.tsx, upcoming.ts, faq.ts, recruitment.ts
- 수정: tailwind.config.ts, globals.css, Button.tsx, Header.tsx, Footer.tsx, HeroSection.tsx, CtaBanner.tsx, projects.ts, announcements.ts, ProjectCard.tsx, About.tsx, router/index.tsx
