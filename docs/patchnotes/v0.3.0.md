# v0.3.0 패치노트

**날짜**: 2026-03-04
**주제**: 패치노트/공지사항 페이지 + UI/UX 접근성 개선

---

## 변경 사항

### 새 페이지
- **패치노트 (/patchnotes)**: 버전별 변경 이력 타임라인, 카테고리 배지 (추가/변경/수정/제거)
- **공지사항 (/announcements)**: 고정 공지 상단 표시, 카테고리 아이콘/배지, 펼치기/접기 인터랙션

### 데이터 자동화
- `constants/patchnotes.ts`: 패치노트 배열에 항목 추가 시 페이지 자동 반영
- `constants/announcements.ts`: 공지사항 배열에 항목 추가 시 페이지 자동 반영
- Header 네비게이션에 공지사항/패치노트 메뉴 추가

### WCAG 2.1 AA 접근성 (KRDS 가이드라인 반영)
- **Skip Navigation**: 본문 바로가기 링크 (키보드/스크린리더)
- **aria-current="page"**: 현재 페이지 네비게이션 표시
- **aria-expanded**: 모바일 메뉴 열림/닫힘 상태 전달
- **aria-label**: 테마 변경, 메뉴 열기/닫기 버튼
- **focus-visible**: 키보드 포커스 시만 2px brand 아웃라인 표시
- **role="main"**: 메인 콘텐츠 영역 시맨틱 마크업

### 한국어 UX 최적화 (Material Design 3 + Apple HIG 참고)
- **Pretendard 폰트**: CDN 서브셋 로딩, 한국어 가독성 최적화
- **word-break: keep-all**: 단어 단위 줄바꿈 (CJK 텍스트 가독성)
- **line-height 1.75**: 한국어 본문 행간 최적화
- **오프화이트 다크 모드 텍스트**: #e2e8f0 (순백 대비 눈 피로 감소)

### 접근성 & 사용성
- **prefers-reduced-motion**: 모션 감소 선호 사용자 대응
- **최소 터치 타겟 44px**: 모든 버튼 min-height 보장 (Apple HIG)
- **index.html**: `lang="ko"`, meta description, 타이틀 PompCore로 변경

### 버그 수정
- **빈 화면 수정**: Supabase 환경변수 미설정 시 앱 크래시 → 안전한 폴백 처리
  - `isSupabaseConfigured` 플래그로 인증 기능 조건부 활성화
  - placeholder URL로 createClient 에러 방지
