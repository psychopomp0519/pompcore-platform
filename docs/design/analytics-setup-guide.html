Analytics & AdSense Setup Guide

PompCore 출시를 위한 분석 도구 및 수익화 연동 가이드

개요 — 연동할 서비스
서비스목적환경변수ID 형식
Microsoft Clarity 히트맵, 세션 리플레이 VITE_CLARITY_ID 영문+숫자 (예: abc123def)
Google Analytics 4 트래픽 분석, 유입 경로 VITE_GA_MEASUREMENT_ID G-XXXXXXXXXX
Google AdSense 디스플레이 광고 수익 VITE_ADSENSE_CLIENT ca-pub-XXXXXXXXXXXXXXXX
모든 서비스는 환경변수 기반 조건부 활성화. 값이 비어있으면 자동으로 비활성화됩니다.
1
Microsoft Clarity 설정

사용자 행동 분석 (히트맵, 세션 리플레이, 분노 클릭 감지)

STEP 1
Clarity 프로젝트 생성
clarity.microsoft.com에 접속하여 Microsoft 계정으로 로그인
→ "Add new project" 클릭
STEP 2
프로젝트 2개 생성
pompcore.cc 메인 사이트용 프로젝트
vault.pompcore.cc Vault용 프로젝트

각 프로젝트 생성 시 사이트 URL에 해당 도메인을 입력
STEP 3
프로젝트 ID 복사
Settings → Overview에서 Project ID를 복사
(영문+숫자 조합, 예: abc123def)
STEP 4
.env에 입력
# 두 앱이 같은 ID를 써도 되고, 별도 프로젝트면 각각 .env에 설정 VITE_CLARITY_ID=your_clarity_project_id
⚠️ Clarity는 18세 미만 사용자를 대상으로 하는 사이트에서 사용할 수 없습니다.
✅ Clarity는 기본적으로 민감한 콘텐츠(입력란, 숫자, 이메일)를 자동 마스킹합니다.
2
Google Analytics 4 설정

트래픽 추적, 유입 경로 분석, 전환 퍼널

STEP 1
GA4 속성 생성
analytics.google.com에 접속
→ 관리(톱니바퀴) → 속성 만들기 → "PompCore" 속성 생성
STEP 2
데이터 스트림 2개 추가
관리 → 데이터 스트림 → "웹" 선택
https://pompcore.cc 스트림 이름: "PompCore Web"
https://vault.pompcore.cc 스트림 이름: "PompCore Vault"

같은 속성(Property) 아래 2개 스트림을 만들면 통합 분석 가능
STEP 3
측정 ID 복사
각 스트림의 세부정보에서 측정 ID를 복사
형식: G-XXXXXXXXXX
STEP 4
.env에 입력
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX
💡 두 앱이 같은 속성의 서로 다른 스트림이라면, 같은 측정 ID를 공유할 수 있습니다. 또는 각각 다른 ID를 사용해도 됩니다.
3
Google AdSense 설정

디스플레이 광고 수익화 (수동 배치 방식)

STEP 1
AdSense 계정 생성 및 사이트 등록
google.com/adsense에 접속
→ Google 계정으로 로그인 → 사이트 URL: pompcore.cc 입력
→ 약관 동의 후 계정 생성
STEP 2
사이트 소유권 확인
AdSense가 제공하는 코드를 사이트 <head>에 추가하여 소유권 확인
(이 구현에서 자동으로 처리됩니다)
STEP 3
게시자 ID 복사
계정 → 계정 정보에서 게시자 ID를 확인
형식: ca-pub-XXXXXXXXXXXXXXXX (16자리 숫자)
STEP 4
광고 단위 생성
광고 → 광고 단위 별 → 디스플레이 광고 선택
→ 광고 단위 이름 입력 (예: "PompCore-Footer")
→ 생성 후 data-ad-slot 번호를 메모
STEP 5
.env에 입력
VITE_ADSENSE_CLIENT=ca-pub-XXXXXXXXXXXXXXXX
⚠️ AdSense 승인에는 며칠~몇 주가 소요될 수 있습니다. 코드는 미리 삽입해두고, 승인 후 자동으로 광고가 표시됩니다.
💡 PompCore는 수동 배치 방식을 사용합니다. 사용자의 핵심 기능 이용을 방해하지 않는 위치(푸터, 섹션 사이 등)에만 광고를 배치합니다.
최종 .env 파일 형태
# Analytics (optional) VITE_CLARITY_ID=abc123def VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX # Monetization (optional) VITE_ADSENSE_CLIENT=ca-pub-XXXXXXXXXXXXXXXX
✅ 모든 값은 선택 사항입니다. 비워두면 해당 서비스가 비활성화됩니다.
