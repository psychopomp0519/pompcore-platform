## 1. PompCore란?

**PompCore**는 서브컬쳐 오픈월드 RPG 게임의 몰입감과 감성을 일상 도구에 접목한 **통합 생활 플랫폼**이다.

> **"일상을 플레이하다 — Play Your Day"**

기존의 딱딱하고 지루한 생산성 도구 대신, 원신·명일방주·블루아카이브 같은 게임의 미학을 차용하여 사용자가 자신의 삶을 하나의 게임처럼 능동적으로 플레이할 수 있게 한다.

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **몰입 (Immersion)** | RPG 세계관과 감성을 일상 도구에 자연스럽게 녹여낸다 |
| **접근성 (Accessibility)** | 무료로 핵심 기능 사용 가능, 유료는 편의성 향상에만 집중 |
| **민첩성 (Agility)** | AI 네이티브 개발으로 빠른 사이클 |
| **공동 성장 (Co-growth)** | 사용자 피드백을 최우선으로 수용 |

## 2. 서비스 라인업

PompCore 생태계의 각 서비스는 RPG 세계관 속 **시설/장소**에 대응한다.

| 서비스 | 영문 | 서브네임 | 역할 | RPG 대응 | 상태 |
|--------|------|----------|------|----------|------|
| **PompCore** | PompCore | 일상을 플레이하다 | 브랜딩 허브, SSO, 서비스 게이트웨이 | 모험가 길드 로비 | 운영 중 |
| **Vault** | Vault | 모험가의 금고 | 개인 재무 관리 (가계부) | 보물 창고 | 운영 중 |
| **Quest** | Quest | 일상이 곧 모험 | 퀘스트 기반 일정 관리 | 퀘스트 보드 | 출시 예정 |
| **Forge** | Forge | 의지의 대장간 | 과업 중심 자기관리 엔진 | 대장간 / 제련소 | 계획 중 |
| **Academy** | Academy | 지식의 길드홀 | 온라인 교육 플랫폼 | 마법 아카데미 | 준비 중 |

### 네이밍 규칙

- 서비스명은 **오픈월드 RPG 세계관에 등장하는 시설/개념**에서 따온다
- 영문 단일 단어, 직관적이고 기억하기 쉬운 것
- 서브네임은 **세계관 몰입형** 스타일 (예: "모험가의 금고", "의지의 대장간")
- 새 서비스 추가 시 동일 규칙을 따른다

## 4. 타이포그래피

| 용도 | 폰트 | 굵기 | 비고 |
|------|------|------|------|
| 디스플레이 (제목, 로고) | **Nunito** | Bold~Black | 라운드형 산세리프, 게임 UI 느낌 |
| 본문 (한글) | **Pretendard** | Regular~Medium | 가독성 최적화, 행간 1.75 |
| 본문 (영문) | **Nunito** | Regular~Medium | 한글과 조화 |
| 숫자 (금액 등) | Pretendard | SemiBold | `tabular-nums` (고정폭) |
| 로고 타이포 | Cinzel 계열 | Bold~Black | 판타지 RPG 세리프 |

### 그라디언트 텍스트

주요 제목에 사용하는 그라디언트:
```css
/* 라이트 모드 */

/* 다크 모드 */
background: linear-gradient(90deg, #A855F7, #FFD700, #EC4899);
```

## 5. 디자인 시스템 (Nebula)

### 5.1 글래스모피즘 카드

모든 정보 카드는 글래스모피즘 스타일을 따른다:
- 반투명 배경 + 블러(16px) + 미세한 보더
- 모서리: `rounded-2xl` (16px)
- 그림자: `shadow-card` (2-12px)
- 호버: 그림자 강화 + 미세한 리프트

### 5.2 배경 장식

**라이트 모드**: 하늘 그라디언트 + 복합 구름 (2-레이어 입체, 드리프트 애니메이션)

**다크 모드**: 밤하늘 + 별 파티클 3티어 + 글로우 오브 3개

### 5.3 애니메이션

| 이름 | 효과 | 시간 |
|------|------|------|
| `twinkle` | 별 깜빡임 (불투명도) | 2s |
| `cloud-drift` | 구름 이동 | 30s |
| `fadeIn` | 등장 (불투명도) | 0.6s |
| `slideUp` | 위로 등장 | 0.4s |
| `float` | 부유 효과 | 3s |

`prefers-reduced-motion` 활성 시 모든 애니메이션 1ms로 비활성화.

### 5.4 아이콘

- 시스템 아이콘: SVG 인라인 (`24×24`, `stroke-width: 2`)
- 서비스 아이콘: 커스텀 SVG 19종 (`@pompcore/ui`의 PlatformIcons)
- 스타일: `stroke` 기반 라인 아이콘, `fill: none`

## 6. 로고 시스템

### 6.1 로고 형태

모든 서비스 로고는 **타이포그래피 워드마크** + **컴팩트 이니셜** 2벌 세트.

| 유형 | 크기 | 용도 |
|------|------|------|
| 워드마크 | 400×120 | 헤더, OG 이미지, 소개 페이지 |
| 컴팩트 | 64×64 | 파비콘, 앱 아이콘, 내비게이션 |

### 6.2 로고 스타일

- **서체**: 판타지 RPG 감성 세리프 (Cinzel 참고)
- **장식**: 서비스 모티프를 글자에 미세하게 반영 (가독성 유지)
- **컬러**: 서비스 고유색 그라디언트 (Primary → Light)
- **포맷**: 순수 벡터 SVG (래스터 임베드 금지)

### 6.3 서비스별 로고 모티프

| 서비스 | 히어로 글자 | 장식 참고 |
|--------|------------|-----------|
| PompCore | **P** | 성운 소용돌이, 마법 에너지 흐름 |
| Vault | **V** | 보석 커팅 면, 열쇠 구멍 |
| Quest | **Q** | 나침반 바늘, 방향 화살표 |
| Forge | **F** | 망치 날, 모루 표면, 각진 마감 |
| Academy | **A** | 깃펜 끝, 마법사 모자, 두루마리 곡선 |

### 6.4 파일 구조


## 7. UI 컴포넌트 가이드

### 7.1 버튼

| 유형 | 스타일 | 용도 |
|------|--------|------|
| Primary | 서비스 컬러 배경 + 흰색 텍스트 + 호버 글로우 | 주요 액션 |
| Outline | 투명 + 서비스 컬러 보더/텍스트 | 보조 액션 |
| Ghost | 텍스트만, 호버 시 배경 | 취소/뒤로 |
| Danger | 빨간색 배경/보더 | 삭제/위험 액션 |

### 7.2 입력 필드

```
rounded-xl border border-navy/10 bg-white/80 px-3 py-2.5 text-sm
focus:border-{service-color} focus:ring-1 focus:ring-{service-color}
dark:border-white/10 dark:bg-white/5
```

### 7.3 모달/다이얼로그

- 배경 오버레이: `bg-black/50` + `backdrop-blur-sm`
- 패널: 글래스 카드 스타일, `max-w-md`, `rounded-2xl`
- 제목 + 본문 + 액션 버튼 구조

### 7.4 토스트/알림

- 하단 우측 고정 (모바일: 하단 중앙)
- 자동 소멸 (5초)
- 유형별 좌측 컬러 라인 (성공: 초록, 오류: 빨강, 정보: 파랑)

## 9. 접근성 기준

- **WCAG 2.1 AA** 준수
- 한글 최소 14px
- 포커스 링: 2px outline, vault-color
- 키보드 네비게이션 완전 지원
- `prefers-reduced-motion` 대응
- 시맨틱 HTML + ARIA 속성
